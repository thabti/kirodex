Kirodex

AI coding agents on your desktop. Chat, plan, diff, commit — all from one native app.

Built with Tauri v2 and React 19. Powered by the Agent Client Protocol. Works with macOS, Linux, and Windows.

Download GitHub

Everything you need to ship with AI

A native desktop app that wraps the Agent Client Protocol into a polished developer experience.

Chat and agents

Threaded conversations with independent agent contexts. Slash commands, @mentions, plan mode, and question cards.

Code and diffs

Syntax-highlighted inline and side-by-side diffs via Shiki. Click any file operation in chat to jump to the diff.

Git integration

Branch, stage, commit, push via libgit2. Worktree support isolates each thread in its own working directory.

Plan mode

Plan before you build. Context-aware handoff suggests switching to implement mode before the context window fills up.

Integrated terminal

Full PTY terminal powered by xterm.js. Run commands without leaving the app. Cmd+L to jump back to chat.

Desktop notifications

Native notifications when the agent finishes while the app is in the background. Configurable in settings.

Install

Available on macOS, Linux, and Windows. Or grab the latest release from GitHub.

Homebrew (recommended)

brew install --cask thabti/tap/kirodex

Or download the .dmg from GitHub Releases.

Debian / Ubuntu

sudo dpkg -i kirodex_*_amd64.deb

AppImage

chmod +x Kirodex_*.AppImage && ./Kirodex_*.AppImage
